# 초등학생용 AI 체험·윤리 교육 프로그램
## MediaPipe × YOLO 기반 10차시 운영안

## 1. 프로그램 개요

| 구분 | 내용 |
|---|---|
| 프로그램명 | 몸으로 배우는 AI 탐험교실 |
| 대상 | 초등 4~6학년 권장 |
| 운영 차시 | 10차시 |
| 권장 시간 | 차시당 40~50분 또는 80분 블록수업 |
| 권장 팀 구성 | 3~4명 |
| 핵심 기술 | MediaPipe Hands / Face / Pose, YOLO Object Detection |
| 수업 성격 | 놀이형, 체험형, 관찰형 |
| 최종 산출물 | 체험 포트폴리오, 윤리 선언문, 팀 아이디어 발표 |

## 2. 운영 목표

| 구분 | 내용 |
|---|---|
| 목표 1 | AI가 사람의 손, 얼굴, 몸, 사물을 어떻게 보는지 쉽고 재미있게 이해한다 |
| 목표 2 | AI도 틀릴 수 있다는 점을 체험으로 배운다 |
| 목표 3 | AI를 사용할 때 지켜야 할 약속과 배려를 익힌다 |
| 목표 4 | 친구들과 협력하여 우리 생활에 도움이 되는 AI 아이디어를 만든다 |

## 3. 평가 방향

| 평가 영역 | 비율 | 평가 내용 |
|---|---:|---|
| 참여 | 40 | 체험 활동 참여도, 미션 수행 태도 |
| 체험 기록 | 20 | 기록지 작성, 관찰 내용 정리 |
| 협업 | 20 | 팀 역할 분담, 친구와의 소통 |
| 발표 | 20 | 결과 설명, 느낀 점 공유 |

## 4. 차시별 운영안

### 1차시. AI야, 내 손 좀 봐!

| 항목 | 내용 |
|---|---|
| 수업 목표 | 손 인식 AI가 손가락과 관절을 점으로 본다는 것을 이해한다 |
| 핵심 활동 | 손 보여주기, 손가락 숫자 맞히기, 가위바위보, 손모양 따라 하기 |
| 기술 활용 | MediaPipe Hands 랜드마크 시연, 손가락 21개 점 관찰 |
| AI 개념 | AI는 손 전체를 이해하는 것이 아니라 특징점을 보고 판단한다 |
| 윤리 포인트 | 카메라 앞 예절, 친구 허락 없이 촬영하지 않기 |
| 결과물 | 손 인식 체험 기록지 |

### 2차시. AI야, 표정을 맞혀봐!

| 항목 | 내용 |
|---|---|
| 수업 목표 | 얼굴 표정 인식이 항상 정확하지 않을 수 있음을 안다 |
| 핵심 활동 | 웃기, 놀라기, 찡그리기, AI가 맞히는지 관찰하기 |
| 기술 활용 | MediaPipe Face 랜드마크 시연 |
| AI 개념 | 얼굴도 눈, 입, 코 주변 특징을 보고 추정한다 |
| 윤리 포인트 | 표정만 보고 기분을 단정하지 않기, 친구 외모 평가 금지 |
| 결과물 | 표정 오인식 모음 카드 |

### 3차시. AI야, 내 몸 따라와!

| 항목 | 내용 |
|---|---|
| 수업 목표 | 자세 인식 AI가 몸의 움직임을 점으로 추적하는 원리를 안다 |
| 핵심 활동 | 한 발 서기, 팔 벌리기, 별 자세 만들기, 몸으로 OX 퀴즈 |
| 기술 활용 | MediaPipe Pose 시연 |
| AI 개념 | 관절의 위치와 움직임으로 자세를 추적한다 |
| 윤리 포인트 | 몸을 놀리거나 비교하지 않기, 신체 차이를 존중하기 |
| 결과물 | 동작 미션 성공표 |

### 4차시. AI는 교실에서 무엇을 찾을까?

| 항목 | 내용 |
|---|---|
| 수업 목표 | YOLO가 사물을 상자와 이름으로 찾는 방식을 이해한다 |
| 핵심 활동 | 교실 물건 탐정 놀이, “AI가 찾은 것 / 못 찾은 것” 비교하기 |
| 기술 활용 | 책상, 의자, 가방, 병, 사람 등 객체 탐지 체험 |
| AI 개념 | 사물 위치를 네모 상자로 표시하고 이름을 붙인다 |
| 윤리 포인트 | 보이는 것만으로 함부로 판단하지 않기 |
| 결과물 | 우리 교실 AI 탐정 노트 |

### 5차시. AI도 틀릴까? 오인식 실험실

| 항목 | 내용 |
|---|---|
| 수업 목표 | 조명, 거리, 가림, 각도에 따라 인식이 달라짐을 체험한다 |
| 핵심 활동 | 모자 쓰기, 물건 일부 가리기, 멀리서 보여주기, 빠르게 움직이기 |
| 기술 활용 | MediaPipe와 YOLO로 같은 대상이 상황에 따라 다르게 인식되는지 확인 |
| AI 개념 | AI 성능은 환경의 영향을 받는다 |
| 윤리 포인트 | AI가 틀렸을 때 사람에게 피해가 생길 수 있음을 이해 |
| 결과물 | 오인식 베스트 컬렉션 |

### 6차시. AI를 어디에 쓰면 좋을까?

| 항목 | 내용 |
|---|---|
| 수업 목표 | 생활 속 문제를 해결하는 AI 아이디어를 떠올린다 |
| 핵심 활동 | 학교, 집, 놀이터에서 불편한 점 찾기, 해결 카드 만들기 |
| 기술 활용 | 손 인식, 자세 인식, 사물 인식 중 적합한 기술 고르기 |
| AI 개념 | AI는 재미뿐 아니라 도움이 되는 도구가 될 수 있다 |
| 윤리 포인트 | 누군가를 감시하거나 불편하게 만들지 않는 방향 생각 |
| 결과물 | AI 활용 아이디어 카드 |

### 7차시. 우리 팀 AI 만들기 회의

| 항목 | 내용 |
|---|---|
| 수업 목표 | 팀별로 하나의 AI 체험 아이디어를 정한다 |
| 핵심 활동 | 팀 이름 정하기, 문제 정하기, 어떤 AI를 쓸지 선택하기 |
| 기술 활용 | MediaPipe 또는 YOLO 중 1개 이상 선택 |
| AI 개념 | 문제와 기술을 연결하는 기획의 기초를 배운다 |
| 윤리 포인트 | 친구들이 안전하고 즐겁게 쓸 수 있는지 점검 |
| 결과물 | 그림 중심 팀 기획서 |
